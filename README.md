## Clone the project

```
$ git clone https://github.com/stonear/go-template
$ cd go-template
```

## REST Resource Naming Guide

Verb | URI | Action
-----|-----|-------
GET | /entities | Index
GET | /entities/create | Create
POST | /entities | Store
GET | /entities/:id | Show
GET | /entities/:id/edit | Edit
PUT | /entities/:id | Update
DELETE | /entities/:id | Destroy
